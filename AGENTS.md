# Project Rules

- This is an existing project. Never scaffold or recreate it.
- Make one small feature at a time.
- Keep implementations simple and readable.
- Do not add dependencies unless they are clearly necessary.
- Do not add Docker, deployment configuration, databases, authentication, AI features, or a backend unless explicitly requested.
- Preserve the existing visual style.
- Before coding, give a brief plan.
- After coding, run the existing lint and build commands, summarize the files changed, explain how I should test the feature, and stop.
